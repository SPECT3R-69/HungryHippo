> To a hungry hippo, everything is a feed

## Running HungryHippo

With docker-compose:

```bash
  docker-compose -f deploy/docker-compose.yml build
  docker-compose -f deploy/docker-compose.yml up
```

Without docker:

```bash
  npm ci
  npm run start
```
